## üî∞ Getting Started

### Prerequisites

- [Flutter](https://flutter.dev/docs/get-started/install)
- [Firebase](https://firebase.google.com/docs/flutter/setup)

### Installation

1. Clone the repo

```sh
git clone
```

2. Install packages

```sh
flutter pub get
```

3. Run the app

```sh
flutter run
```
